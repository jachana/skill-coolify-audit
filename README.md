# coolify-audit

Read-only audit of a Coolify instance — full resource inventory, env-var hygiene (plaintext secrets), exposed ports, and servers that need validation

## Install (Claude Code)

```
/plugin marketplace add jachana/claude-marketplace
/plugin install coolify-audit@jachana
```

Part of the [jachana marketplace](https://github.com/jachana/claude-marketplace).
